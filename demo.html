🧠 NotebookLM API Deep Dive

🔑 Key Insight: NotebookLM provides contextualized AI analysis grounded in your sources. Unlike generic LLMs, it cites sources and maintains context across documents.

Why NotebookLM?

  • Source-Grounded AI: Answers are based on your specific content, not generic training data
  • Citation Support: Responses include references to source material
  • Context Management: Maintains conversation history across queries
  • Multi-Format Support: YouTube videos, PDFs, web pages, Google Drive files
  • Free Tier: ~50 queries/day perfect for POC development

List all notebooks. Add --json for programmatic parsing.
nlm notebook list --json

Create a new notebook with specified title.

nlm notebook query
Query notebook with a prompt. Returns JSON with answer, sources, and conversation ID.
nlm notebook query <notebook-id> "Summarize the key points"
nlm notebook delete
Delete a notebook permanently. Use --confirm to skip confirmation.
nlm notebook delete <notebook-id> --confirm

Permanently removes a source from the notebook. Use --confirm to skip confirmation prompt.

Get raw content of a source (no AI processing).

NotebookLM Response Structure

When you query NotebookLM, you get a JSON response like this:

{ "value": { "answer": "This video discusses three main topics: 1) Market trends showing increased volatility, 2) Investment strategies for risk mitigation, and 3) Predictions for Q2 performance based on historical data.", "conversation_id": "abc123-def456-ghi789", "sources_used": [ { "id": "source-xyz", "title": "Q1 Market Analysis Video" } ] } }

Our Implementation: We parse this JSON and extract response["value"]["answer"] for clean email delivery.

Source Management Strategy

🎯 Why We Clear Sources Before Each Video

NotebookLM analyzes ALL sources in a notebook. To ensure focused analysis on a single video:

  1. List Sources: Get all current source IDs
  2. Delete Each: Remove old sources one by one
  3. Add New: Add fresh video as the only source
  4. Query: Get AI analysis of just that video

Result: Clean, focused insights without context pollution from old sources.

Module Breakdown

📁 src/config.py

Purpose: Centralized configuration management

  • Loads environment variables from .env file
  • Validates required API keys and credentials
  • Provides default values for optional settings

📁 src/youtube_monitor.py

Purpose: YouTube API integration

  • Multiple channel lookup methods (forHandle, forUsername, search)
  • Fetches latest videos with metadata
  • Handles Unicode in video titles safely

📁 src/state_manager.py

Purpose: Tracks processed videos

  • JSON file storage with FileLock for concurrency
  • Prevents reprocessing same video multiple times
  • Stores video ID, title, timestamp, and status

📁 src/notebooklm_handler.py

Purpose: NotebookLM CLI wrapper (⭐ Core AI Module)

  • list_sources(): Get all sources in notebook
  • delete_source(): Remove a specific source
  • clear_all_sources(): Remove all sources
  • add_source(): Add YouTube video (auto-clears first)
  • query_notebook(): Get AI analysis with JSON parsing
  • ensure_authenticated(): Auto-login when needed

📁 src/email_sender.py

Purpose: Email notification system

  • Professional HTML template with purple gradient
  • Multi-recipient support (comma-separated)
  • Plain text fallback for email clients
  • SMTP SSL connection for Gmail

📁 src/main.py

Purpose: Main orchestration

  • Initializes all modules with configuration
  • Fetches latest videos → Filters new ones → Processes each
  • Error handling with graceful degradation
  • Logging to files with UTF-8 encoding

💻 Key Code Implementations

1. NotebookLM Source Clearing

def clear_all_sources(self) -> bool: """Remove all sources from the notebook.""" logger.info("Clearing all sources from notebook") # Get list of all sources sources = self.list_sources() if sources is None: logger.error("Failed to get source list") return False if len(sources) == 0: logger.info("No sources to clear") return True # Delete each source all_deleted = True for source in sources: source_id = source.get('id') if source_id: if not self.delete_source(source_id): all_deleted = False else: time.sleep(1) # Small delay between deletions return all_deleted

2. JSON Response Parsing

def query_notebook(self, prompt: str) -> Optional[str]: """Query the notebook and extract clean answer.""" command = ["nlm", "notebook", "query", self.notebook_id, prompt] success, output = self._run_command(command) if success: try: # Parse JSON response from NotebookLM CLI response_data = json.loads(output) # Extract the answer from nested structure if isinstance(response_data, dict) and "value" in response_data: answer = response_data["value"].get("answer", "") logger.info(f"Extracted answer ({len(answer)} characters)") return answer else: logger.warning("Response not in expected format") return output except json.JSONDecodeError as e: logger.warning(f"Could not parse JSON: {e}") return output return None

3. UTF-8 Encoding Handling

def _run_command(self, command: list) -> Tuple[bool, str]: """Run CLI command with proper encoding.""" result = subprocess.run( command, capture_output=True, text=True, encoding='utf-8', errors='replace', # Replace problematic chars timeout=120, env={**os.environ, "PYTHONIOENCODING": "utf-8"} ) if result.returncode == 0: return True, result.stdout.strip() else: return False, result.stderr.strip()

4. State Management with FileLock

def mark_processed(self, video_id: str, title: str, published_at: str, notebook_id: str, status: str = "success") -> bool: """Mark a video as processed with file locking.""" with FileLock(self.lock_file): state = self._load_state() state["processed_videos"][video_id] = { "title": title, "published_at": published_at, "processed_at": datetime.now().isoformat(), "notebook_id": notebook_id, "status": status } self._save_state(state) logger.info(f"Marked video as processed: {video_id}") return True

5. Multi-Recipient Email

# In .env file: EMAIL_RECIPIENT=rahulsaigsr@gmail.com,thodupunuriharshith0699@gmail.com # Email handler automatically processes comma-separated list msg["To"] = recipient_email # Can contain multiple addresses with smtplib.SMTP_SSL(self.smtp_server, self.smtp_port) as server: server.login(self.sender_email, self.sender_password) server.send_message(msg) # Sends to all recipients

6. Retry Logic with Exponential Backoff

for attempt in range(self.max_retries): try: result = subprocess.run(command, ...) if result.returncode == 0: return True, result.stdout else: # Check if authentication error if "authentication" in result.stderr.lower(): self._authenticate() continue # Exponential backoff if attempt < self.max_retries - 1: time.sleep(self.retry_delay * (attempt + 1)) except Exception as e: logger.error(f"Error: {e}") continue
